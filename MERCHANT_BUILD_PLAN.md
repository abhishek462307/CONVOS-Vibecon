# Merchant Panel Build Plan

## Features to Implement

### 1. Orders Management ✅
- View all orders with filtering
- Update order status (pending → processing → shipped → delivered)
- Update payment status
- Add tracking numbers
- View order details
- Export orders

### 2. Product Catalog Management ✅
- Add new products
- Edit existing products
- Delete products
- Bulk actions
- Stock management
- Price updates

### 3. Customers Management ✅
- View customer profiles
- See purchase history
- View trust scores
- Send messages
- Block/unblock customers

### 4. Shipments Tracking ✅
- View all shipments
- Add tracking numbers
- Update carrier info
- Change shipment status
- Print shipping labels

### 5. Reviews Moderation ✅
- Approve/reject reviews
- Reply to reviews
- Delete spam
- Feature reviews

### 6. Campaigns Management ✅
- Create new campaigns
- Edit campaigns
- Set audience
- Schedule sends
- View analytics

### 7. Store Design ✅
- Edit store name
- Update tagline
- Change AI settings
- Upload images
- Theme colors

### 8. Live Intent Stream ✅
- Real-time buyer actions
- Filter by type
- Action alerts
- Quick actions

### 9. Statistics Dashboard ✅
- Revenue metrics
- Order analytics
- Customer metrics
- Product performance
- Real-time updates

## Implementation Order
1. Complete API endpoints for CRUD operations
2. Build UI components for each section
3. Add forms with validation
4. Implement real-time updates
5. Add export/print functionality
6. Polish UI/UX
